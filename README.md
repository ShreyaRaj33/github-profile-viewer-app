# 🚀 GitHub Profile Viewer

A simple and interactive web application that allows users to search for any GitHub profile and view key details in a clean 

## 🔥 Features
- 🔍 Search any GitHub username  
- 👤 View profile picture, name, and bio  
- 📊 Display followers, following, and public repositories  
- 📁 View top repositories with links  
- ⚡ Fast and responsive UI  
- ❌ Error handling for invalid usernames  


## 🛠️ Tech Stack & Concepts Used

### 🔹 HTML
- Semantic structure (`div`, `input`, `button`)
- Form handling for user input
- Dynamic content placeholders

### 🔹 CSS
- Flexbox for layout  
- Custom styling (colors, spacing, typography)  
- Card-based UI design  
- Responsive design basics  

### 🔹 JavaScript
- DOM Manipulation (`getElementById`, `innerHTML`)  
- Event Handling (`click` event listener)  
- Async/Await for API calls  
- Fetch API for HTTP requests  
- Error Handling (`try...catch`)  
- Array methods (`map`, `slice`) for rendering repos
